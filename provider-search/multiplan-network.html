Member Services - Provider Search - Multiplan Network

Multiplan Network - https://www.multiplan.com

1. Open Provider Search

○ Click the Provider Search tab in your browser.

2. Select Network

○ Click on Multiplan Provider Search.

3. Choose Network

○ Click Select Network on the left-hand side.

○ Then select Multiplan.

4. Select Plan Type:

○ Click on Limited Benefit Plan.

5. Enter Search Criteria:

○ Type in the provider's name, facility, NPI, or specialty in the search box.

6. Verify Captcha:

○ Complete the captcha verification ("I’m not a robot").

7. Search

○ Click the magnifying glass icon to initiate the search.
