Step 1: Verify Member Plan

Check Member Profile:
○ Access the member’s profile to verify which plan they have.

Step 2: Conduct Provider Search for Elite MD Policies

1. Open Provider Search:

○ Click the Provider Search tab in your browser.

2. Select Network:

○ For Elite MD policies, click on First Health.

3. Choose Network:

○ Select First Health Network.

4. Start the Search:

○ Click Start Now.

5. Select Provider Type:

○ Choose the appropriate provider type from the options provided.

6. Enter Zip Code:

○ Type in the relevant zip code to locate providers in that area.

7. More Options (if needed):

○ For a more detailed search, click on More Search Options.
